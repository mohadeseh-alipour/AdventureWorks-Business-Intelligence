# 📊 Business-Intelligence-PowerBI-Dashboard
An interactive **Power BI** dashboard built using the AdventureWorks sample dataset to analyze sales performance, customer behavior, product performance, and regional trends.

The project demonstrates data modeling, DAX calculations, interactive reporting, and business intelligence techniques to transform raw data into meaningful insights.

## 🎯 Project Objectives
This dashboard enables users to:

- Monitor key business KPIs
- Analyze sales trends over time
- Identify top-performing products
- Explore customer purchasing behavior
- Compare regional sales performance
- Support business decision-making through interactive reports

## 🛠 Tools & Technologies
- Power BI Desktop
- DAX
- Power Query
- Star Schema Data Modeling
